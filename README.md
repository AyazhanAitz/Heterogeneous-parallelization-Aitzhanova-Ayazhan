# Последовательная и параллельная обработка массивов (C++ и OpenMP)

## Описание

Данная программа демонстрирует **последовательную и параллельную обработку больших массивов в C++** с использованием библиотеки **OpenMP**.  
В программе динамически выделяется память под массивы, они заполняются случайными числами, вычисляются среднее значение, минимум и максимум, а также проводится **сравнение времени выполнения** последовательных и параллельных алгоритмов.

Цель работы — изучить:
- основы параллельных вычислений с OpenMP;
- использование `reduction` и `critical`;
- влияние параллелизации на производительность при работе с большими массивами.

---

## Используемые технологии

- **C++**
- **OpenMP**
- `<chrono>` — измерение времени выполнения
- Динамическое выделение памяти (`new[]`, `delete[]`)

---

## Структура программы

### Основные функции

- `fillRandom(int* arr, int n)`  
  Заполняет массив случайными целыми числами в диапазоне **от 1 до 100**.

- `averageSequential(const int* arr, int n)`  
  Вычисляет среднее значение элементов массива **последовательно**.

- `averageParallelOMP(const int* arr, int n)`  
  Вычисляет среднее значение **параллельно** с использованием OpenMP и `reduction`.

- `minMaxSequential(const int* arr, int n, int& mn, int& mx)`  
  Находит минимум и максимум в массиве **последовательно**.

- `minMaxParallelOMP(const int* arr, int n, int& mn, int& mx)`  
  Находит минимум и максимум **параллельно** с использованием OpenMP (`for`, `critical`).

- `usNow()`  
  Возвращает текущее время в **микросекундах** для измерения производительности.
